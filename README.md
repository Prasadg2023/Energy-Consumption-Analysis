# 🌍 Global Energy Consumption Analysis Using SQL

A data analysis project focused on understanding global energy consumption, emissions, GDP, and population trends using SQL.

---

## 📌 Project Overview

This project analyzes worldwide energy data to uncover patterns in:

- ⚡ Energy Consumption & Production  
- 🌱 Carbon Emissions  
- 📈 Economic Growth (GDP)  
- 👥 Population Trends  

Using SQL queries, the project explores relationships between these factors and provides meaningful insights for real-world decision-making.

---

## 🎯 Objectives

- Analyze global energy consumption trends  
- Compare energy usage across countries  
- Study the relationship between GDP, population, and energy demand  
- Evaluate carbon emissions and environmental impact  

---

## 🗂️ Database Schema

The project uses the following tables:

- **Country** – List of countries  
- **Emission** – Carbon emissions data  
- **Population** – Population statistics  
- **GDP** – Economic data  
- **Consumption** – Energy consumption  
- **Production** – Energy production  

All tables are connected using the **country** field.

---

## 🔍 Key SQL Concepts Used

- SELECT, WHERE, ORDER BY  
- GROUP BY & HAVING  
- Aggregate Functions (SUM, AVG, MAX)  
- JOINs (INNER JOIN, LEFT JOIN)  
- Subqueries  

---

## 📊 Key Insights

- 🌏 China, USA, and India are the top energy consumers and emitters  
- 📈 Global emissions are steadily increasing  
- 💰 Higher GDP leads to higher energy consumption  
- 👥 Population growth significantly impacts energy demand  
- 🔥 Fossil fuels (coal & oil) contribute most to emissions  

---

## 🛠️ Tools & Technologies

- SQL (MySQL / PostgreSQL)  
- Data Analysis Techniques  
- Relational Database Design
